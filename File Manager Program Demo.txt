                                           **File Manager**

Quick Info:

Remember to press the Enter Key at the end of your answer


Program Description:

In this program you can create folders and files (with extensions e.g .txt).

You can also delete them, rename them and display them.

The maximum number of folders allowed is 10.


The maximum number of files in a folder allowed is 100.




******************************************************************************************************


                                             Instructions:



                               To create a folder: type 'create folder'


                                 To create a file: type 'create file'


                               To delete a folder: type 'delete folder'


                                 To delete a file: type 'delete file'


                               To rename a folder: type 'rename folder'


                                 To rename a file: type 'rename file'


                           To sort any part of the file manager: type 'sort'


                          To display all the folders:  type 'display folders'


                      To display all the files in a folder: type 'display files'


                            To empty the entire file manager: type 'empty'


                                   To exit the program: type 'exit'


                        You can always type 'help' to display the instructions.
